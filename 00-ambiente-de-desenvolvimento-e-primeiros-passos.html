Configurando ambiente Instalando na máquina Windows: https://www.python.org/downloads/ para saber se instalar: python --version Linux e MacOS: Normalmente python já vem instalado, para poder verificar utilizar comandos como: python -V python3 -V Baixando e Configurando IDE para desenvolvimento: Utilizaremos o VSCODE, gratuito e suporta várias tecnologias: https://code.visualstudio.com/download Após instalar, procurar extensão do python e Visual studio intellicode e habilitar.
