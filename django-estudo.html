Instalar python criar ambiente virtual: Na pasta dp projeto: o VENV tem a responsabilidade de isolar o projeto do sistema operacional. para criação do ambiente: python -m venv nome_da_venv Nisso vai criar uma pasta com nome_da_venv e dentro os módulos. E para ativar: No windows: source venv/Scripts/activate ou .\activate No linux: source venv/bin/activete Instalando pacote django: pip install django criando projeto django: django-admin startproject app . (o ponto final é pra ele não criar outra pasta) Dentro da pasta app vai tá as configurações do projeto django. para rodar o sistema: python manage.py runserver
